Test 12 Privacy Policy

Last updated: 2025-12-28

Overview

Test 12 represents apps as metadata and store links. Participation is not enforced. This policy explains what we collect and how it is used.

Data We Collect

  • Submission metadata: app name, store link, and user identifier.
  • Basic operational state needed to run the queue/session logic.

We do not upload app binaries. We do not require installs, reviews, or feedback forms.

How We Use Data

  • Maintain a chronological queue.
  • Run 14-day sessions of exactly 12 apps when a session opens.
  • Support abuse handling via admin actions (when applicable).
